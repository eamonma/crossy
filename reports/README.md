# Frozen extraction reports

`v2-spec-extraction.md` and `v3-mining.md` land here verbatim when recovered: exact
constants (pixel values, timings) and history mined from the previous systems.
Reference only — DESIGN.md stands alone and nothing in the build requires these files
(DESIGN.md preamble).

`spikes/` holds the written answers from the spike track (ROADMAP.md): one short
markdown file per spike, stating the question, what was tried, and the answer. Spike
code itself is throwaway and never merges; these notes are what survives.
